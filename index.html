Introduction

About Moodie

Moodie lets users record their daily emotions, monitor mood patterns, and reflect on their mental well-being over time — built with Jetpack Compose and modern Android architecture.

Supporting SDG 3: Good Health and Well-Being

Moodie promotes mental wellness by enabling users to track, understand, and reflect on their emotions regularly. This application aligns with SDG 3 by providing a simple yet powerful tool to support mental health awareness and emotional well-being.

Labs

Lab 1: UI Design

Understanding Jetpack Compose layouts and building the foundation of Moodie's user interface.

Reflection: This lab helped me understand the basics of Jetpack Compose layouts through the development of Moodie's home screen. I applied Column, Row, and Box components to arrange interface elements in a structured and user-friendly way. The experience improved my understanding of UI design and prepared me for future app development tasks.

Lab 2: Interaction & State

Adding interactive elements and managing state with TextField, Button, and state management.

Reflection: In Lab 2, I enhanced Moodie by adding interactive features using TextField, Button, and state management with remember and mutableStateOf. Users can enter information and receive immediate feedback on the screen, making the application more engaging and responsive. Through this lab, I gained a better understanding of how state works in Jetpack Compose.

Lab 3: Material Design

Applying Material Design principles and creating a visually appealing interface with custom themes.

Reflection: In Lab 3, I improved the Moodie application by applying Material Design principles, including a custom theme, cards, and animations. I selected colors that reflect mental wellness and created a more visually appealing interface using Card components to organize content. This lab helped me understand how themes and animations enhance user experience.

Lab 4: Navigation

Implementing multi-screen navigation and managing data flow across screens with ViewModel.

Reflection: In Lab 4, I enhanced Moodie by implementing multiple screens using Jetpack Compose Navigation and managing user data with ViewModel. The app was upgraded to allow users to navigate smoothly between different pages while maintaining their mood-related information across screens. I gained a better understanding of application architecture and state management.

Lab 5: Room Database

Integrating persistent storage with Room Database to save and retrieve mood records.

Reflection: In Lab 5, I integrated Room Database into Moodie to enable persistent storage of users' mood records. By implementing Entity, DAO, Repository, ViewModel, and Room Database components, I was able to save and retrieve mood data even after the application was closed and reopened. This lab helped me understand the MVVM architecture and data persistence.

Learning Journey & Reflection

Progress and Growth

Throughout this course, I progressed from designing a simple static interface in Lab 1 to developing Moodie, a fully functional mobile application with navigation, persistent storage, cloud integration, APIs, and sensor features. Each lab introduced new concepts that gradually strengthened my skills in Jetpack Compose, state management, Material Design, ViewModel, and Room Database.

Key Challenges and Solutions

One of the biggest challenges was integrating multiple technologies into a single application. Managing data flow across different screens, persisting data with Room Database, and synchronizing with Firebase required careful planning and understanding of the Android architecture components. However, overcoming these challenges improved my problem-solving and development skills significantly.

Impact and Future Direction

Overall, this learning journey has deepened my understanding of Android application development while allowing me to create a solution that supports SDG 3: Good Health and Well-Being. Moving forward, I am excited to explore more advanced features such as machine learning integration, advanced UI/UX patterns, and backend optimization to further enhance Moodie as a tool for supporting mental wellness.
